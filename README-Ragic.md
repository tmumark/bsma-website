# 案件查詢 — Ragic 設定教學（正式方案）

本網站**採用 Ragic** 作為案件查詢後端。協會人員在 Ragic 維護案件，網站的查詢頁直接嵌入 Ragic 查詢介面。

| 頁面 | 檔案 | 用途 |
|------|------|------|
| 正式案件查詢頁 | [search.html](search.html) | 嵌入 Ragic 查詢介面（導覽「案件查詢」指向此頁） |
| 展示版（備用） | [search-sheet.html](search-sheet.html) | 用範例資料示範查詢長相，方便你在設定 Ragic 前先看效果，正式上線可不對外連結 |

> 在你把 Ragic 嵌入網址填進 `js/config.js` 之前，[search.html](search.html) 會顯示「設定步驟」指引；想先看查詢互動長相，開 [search-sheet.html](search-sheet.html)。

---

## 為什麼選 Ragic（重點回顧）

- 台灣公司、繁體中文、非工程師可自行維護（像填 Excel）。
- 內建公開查詢嵌入元件 + **精確比對（Exact Mode）**，民眾輸入完整條件才查得到該筆，**個資保護好**且不必把整張表對外公開。
- 能手動或**定期從網址自動匯入 CSV** —— 這正對應「未來政府給 CSV、但沒有開放 API」的現實（台灣公安申報系統 2026 現況無對外 API）。

---

## 設定步驟（5 步，約 10 分鐘）

### 步驟 1：建立帳號與「案件」表單
1. 到 [ragic.com（繁中）](https://www.ragic.com/intl/zh-TW) 註冊（可先用免費版或 30 天專業版試用）。
2. 新建一張表單，命名「**案件**」。
3. **最快做法**：用「**匯入 → Excel/CSV**」直接上傳專案內的 **`案件範例匯入.csv`**，Ragic 會自動依第一列建立以下欄位並帶入 8 筆範例資料：

   `收件日期`、`收件文號`、`申報建築物名稱`、`申報類別`、`行政區`、`地址`、`目前進度`、`審查結果`、`更新日期`、`備註`

   > 匯入時編碼請選 **UTF-8**（避免中文亂碼）；並把「收件文號」設為 **Key（唯一辨識欄位）**，方便日後更新覆寫。

### 步驟 2：個資保護設定（重要）
- 把 `申報人姓名`、`電話`、`身分證`等**個資欄位設為內部欄位 / 限制存取**，不對外顯示。
- 對外只開放：`收件文號`、`收件日期`、`申報建築物名稱`、`行政區`、`地址`、`目前進度`、`審查結果`、`更新日期`。

### 步驟 3：取得嵌入碼
1. 開啟「案件」表單 → 上方 **「工具」→「在你的網站嵌入此表單」**。
2. 元件類型選 **「資料庫查詢瀏覽（Query / View & Query）」**。
3. 開啟 **「精確比對（Exact Mode）」**：民眾需輸入完整查詢值（如完整收件文號）才查得到該筆 → 防止有人列出全部案件、保護個資。
4. Ragic 會給一段 JavaScript 嵌入碼，裡面有：
   ```js
   var ragic_url = 'ap16.ragic.com/你的帳號/-/編號';
   var ragic_feature = 'query';
   var exactMatch = true;
   ```

### 步驟 4：貼進設定檔
打開 [js/config.js](js/config.js)，把上面的值填進 `ragic`：

```js
ragic: {
  url:        "ap16.ragic.com/你的帳號/-/編號",  // 嵌入碼中的 ragic_url
  feature:    "query",
  exactMatch: true,
  showSheetName: true
},
```

### 步驟 5：開放匿名存取（重要！）
嵌入碼能載入，但若那張表單**沒有開放未登入存取**，民眾打開會看到 **Ragic 登入頁**。請在 Ragic：
- 開啟該表單 → **共享 / 權限設定** → 允許**未登入者（公開 / Everyone）**存取，或確認 **Exact Mode 查詢**對未登入者開放。
- 自我檢查：用**無痕視窗**（未登入狀態）直接開該表單網址，若被導到登入頁＝尚未開放，需調整權限。

### 步驟 6：完成
存檔後重新整理 [search.html](search.html)，即顯示 Ragic 查詢介面。日後新增／修改案件只要在 Ragic 後台操作，網站即時反映。

## （選用）未來串接政府 CSV
若機關同意定期提供一個**公開可下載的 CSV 連結**：
- 在 Ragic 表單 →「**定期從網址匯入**」→ 貼上該 CSV 網址 → 設定每日自動同步（以「收件文號」為 Key 覆寫更新）。
- 這是現況下最務實的「半自動串接政府資料」方式（政府端無開放 API）。

## 注意事項
- **免費版限制**：3 張表單、每張 1,000 筆、10GB。超過需升級（專業版約 US$19/人/月，表單與筆數不限；對外查詢的民眾不佔付費席次）。另有「社群版」永久免費但**整庫須對公眾公開**，僅適合只放公開欄位者。
- 採購前可向 Ragic 確認「**伺服器是否在台灣在地化**」與 ISO 27001 證書效期（官方頁未明示在地化主機）。
- 對外查詢頁建議標明「**資料更新時間**」與「**以主管機關正式查詢為準**」之免責聲明。
- 若日後希望查詢結果外觀**完全比照本站風格**（而非 Ragic 內建外觀），需加一個小後端 proxy 讀 Ragic API（Ragic API 需驗證且瀏覽器跨網域被擋）。可用免費的 Cloudflare Workers，屆時我可再協助。
